# 🐦 Flappy Bird Game

A Flappy Bird clone built using **Java Swing** and **AWT Graphics**. This project recreates the classic Flappy Bird gameplay where players control a bird, avoid obstacles, and try to achieve the highest score possible.

## 🎮 Features

* Smooth bird movement with gravity physics
* Randomly generated pipes
* Collision detection system
* Real-time score tracking
* Game Over screen
* Restart functionality
* Lightweight desktop application

## 🛠️ Tech Stack

* Java
* Java Swing
* AWT Graphics
* Object-Oriented Programming (OOP)

## 🚀 Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/Oswin-Ranjan/FlappyBirdGame.git
cd FlappyBirdGame
```

### Compile the Project

```bash
javac App.java FlappyBird.java
```

### Run the Game

```bash
java App
```

## 🎯 How to Play

* Press **Spacebar** to make the bird jump.
* Avoid hitting the pipes.
* Avoid falling to the ground.
* Pass through pipe gaps to increase your score.
* Press **Spacebar** after Game Over to restart.
